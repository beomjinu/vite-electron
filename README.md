# vite-electron
Simple Vite + Electron template.

<img src="/blob/npmrundev.gif" width="80%" />

## Quick Setup
```bash
# clone this repo
git clone https://github.com/beomjinu/vite-electron.git

# enter project
cd vite-electron

# install dependency
npm install

# develop
npm run dev

# build
npm run build
```
